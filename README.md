# EPSI Shop

Application e-commerce Flutter utilisant l'API FakeStore.

## Prérequis

* Flutter SDK
* Chrome (pour le web)

## Installation

* Cloner le projet
* `flutter pub get`

## Lancement

* `flutter run -d chrome`

## Fonctionnalités

* Liste des produits
* Détail des produits
* Panier avec TVA
* Gestion des quantités
